● Resend hat nichts mit Supabase zu tun — das ist ein separater E-Mail-Dienst für das Kontaktformular.

  Du hast zwei Optionen:

  Option A — Resend-Account erstellen (kostenlos)
  - resend.com → Sign up → "API Keys" → "Create API Key"
  - Kostenlos bis 3.000 Mails/Monat
  - Den Key dann in Vercel als RESEND_API_KEY eintragen

  Option B — Resend weglassen (für jetzt)
  - Lass RESEND_API_KEY in Vercel einfach leer
  - Das Kontaktformular zeigt dann eine Fehlermeldung wenn jemand abschickt
  - Kann später jederzeit nachgetragen werden

  Für den Launch reicht Option B völlig — die Domain, die Seiten und die Supabase-Daten funktionieren alle ohne Resend.
